# 关于本游戏

《植物大战僵尸Python版》是一款由一位12岁的男孩在~~吃饱了没事干~~获得一个灵感后开发出的一款植物大战僵尸改版，本游戏完全使用Python语言编写，且永久免费，源代码于Github上开源 

### 植物大战僵尸Python版与植物大战僵尸其他改版有什么不同点？

下表列出了《植物大战僵尸》改版与《植物大战僵尸》Python版的不同点

|      | 植物大战僵尸改版              | 植物大战僵尸Python版           |
| ---- | --------------------- | ----------------------- |
| 编写流程 | 通过更改游戏16进制数值实现，较为繁琐   | 完全使用Python语言编写，后续更新简单   |
| 灵活度  | 更改时需翻看数值，两个版本发布日期可达数月 | 更新时只需更改源代码，实现快速更新与修复BUG |
| 更新频率 | 数月一次                  | 30-50日发布一小版本，大版本约几个月一个  |
